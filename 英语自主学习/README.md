听力每课只有一份卷子~但是综合会有6份……比较简单的方法是开着Fiddler抓包，在开学习程序的时候会有一个ashx的页面被抓到，看看里面的题目序号就知道了

要是你实在不知道什么是Fiddler……其实其他的抓包工具也是可以的……但是你要是不知道我在说啥……私以为你还是比对六份答案和题目然后找个靠谱的吧
